📦 FIRST TIME SETUP (Do this once):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

OPTION A: Automatic Installation (EASIEST)
   1. Double-click "INSTALL.bat"
   2. Wait for installation to complete
   3. Press Y when asked to start the app
   4. Done! ✅

OPTION B: Manual Installation
   1. Press Windows + R
   2. Type: cmd
   3. Press Enter
   4. Type: cd Desktop\craftly-ai-complete
   5. Type: npm install
   6. Wait 2-5 minutes
   7. Type: npm start
   8. Done! ✅


🎯 RUNNING THE APP (After first setup):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
